# 2048 (Vanilla JS)

Простая версия игры **2048** на HTML/CSS/JavaScript.

## Как запустить

### Вариант 1: просто открыть файл
1. Скачайте/склонируйте проект.
2. Откройте `index.html` в браузере.

### Вариант 2: через локальный сервер (рекомендуется)
```bash
python3 -m http.server 8080
```

После этого откройте в браузере: <http://localhost:8080>

## Управление
- Стрелки клавиатуры
- WASD
- Свайпы на мобильных устройствах

## Возможности
- Поле 4x4
- Счёт и рекорд (сохраняется в `localStorage`)
- Определение победы (2048) и поражения (нет ходов)
- Кнопка «Новая игра»
